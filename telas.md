# Telas criadas na Primeira Etapa do Projeto: GUI Amigável

## 1. Tela de Login e Sign Up

Ao iniciar o programa, o usuário se depara com a tela de Login. Aqui também é possível cadastrar uma nova senha, caso tenha esquecido.

<p align="center">
  <img width="1290" alt="image" src="https://github.com/user-attachments/assets/6405bfd5-45a8-4671-91f3-8293f533c352" />
  <br>
  <em>Figura 1.1: Tela inicial de Login.</em>
</p>

Caso o usuário não possua conta cadastrada no sistema, ele poderá clicar o botão **"Crie Aqui"** para criar uma nova conta. O botão invoca a seguinte tela:

<p align="center">
  <img width="1250" alt="image" src="https://github.com/user-attachments/assets/0b467e9a-a400-42e5-832d-21125391e2fc" />
  <br>
  <em>Figura 1.2: Tela de SignUp do usuário.</em>
</p>

Aqui, o usuário escolherá o tipo de conta que criará (que será importante para acessar funcionalidades futuras dentro do programa), e fará todo o processo de cadastramento usual.

Em seguida, o usuário retornará para a Tela de Login e poderá clicar o botão **"Login"**. Isto fechará a Tela de Login e o usuário será redirecionado para a Tela Principal do programa.

> **Nota:** Seção criada por João Guilherme Possati.

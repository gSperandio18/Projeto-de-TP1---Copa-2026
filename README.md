# Projeto-de-TP1---Copa-2026
Programa em Java, com interface amigável ao usuário criada com Swing, com o intuito de facilitar o gerenciamento da Copa do Mundo de 2026.

Este programa foi criado para cumprir o trabalho acadêmico final da matéria de Técnicas de Programação 1, ministrado pela Dra. Prof. Roberta Barbosa Oliveira, da Universidade de Brasília. A IDE utilizada foi o Netbeans com JDK 25 e Maven instalados.
